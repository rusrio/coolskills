# Codex Skills

A repository of reusable skills for Codex.

This repo stores versioned skills in one place so they can be maintained, improved, and shared with a clear structure.

## Structure

```text
skills/
  <skill-name>/
    SKILL.md
    agents/
      openai.yaml
```

## Purpose

- Keep custom Codex skills in one place
- Organize them in a simple folder structure
- Version and improve them over time

## Current Skills

- `strategic-naming`: evaluates, explores, refines, and replaces names for products, brands, apps, and projects
- `hytale-modding`: creates, modifies, and debugs Hytale Java plugins, ECS systems, commands, and asset-pack content
