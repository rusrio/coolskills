# Hytale Documentation Sources

Use the inline content in `SKILL.md` first. Fetch raw documents only when the user asks about an area that is not already covered well enough.

## Usage Policy

- Prefer the guidance already in the skill for commands, events, ECS basics, permissions, and asset-pack setup.
- Fetch only the specific raw document needed for the task.
- Do not invent Java APIs. If the answer is not in the skill or in the relevant document, say so clearly.
